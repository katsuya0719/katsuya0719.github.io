Eppy - scripting language for EnergyPlus

Posted on 4æœˆ 03, 2017 in Simulation

Simulation requires many data processing task

One of my daily job is doing energy simulation for building. Using energy simulation, we can evaluate and imporve energy performance visualizing and investigating the result. However,it is not that easy to contribute to design. One of the reason is it takes time to go through energy simulation process. Thus, I think it is quite important to shorten the time for energy simulation process in order to contribute to better design decision.

Eppy can automate post processing work

Eppy is python library for procesing data related to EnergyPlus. After doing energy simulation, energyplus produce lots of data as HTML file format.(Part of the example file is shown in below. This is just 1 page of total 107 pages!)
Conventionally, we have been checking the result extracting parameters such as energy consumption, heat balance from this html file which was time-consuming.

One of the function of eppy is parsing this html file. So once we specify the parameter we want to know, eppy extract that from the html file.Using this function, we can reduce post processing time and dig into the data more.
